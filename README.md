# Gather

Plan things with your people.

**Live:** [gather-nine-umber.vercel.app](https://gather-nine-umber.vercel.app)

---

## What it does

- **Create events** — describe an event in plain English and Claude parses it into a structured form
- **Vote on times & locations** — friends vote on options; the event finalizes once there's a majority
- **Map view** — finalized events appear as pins on an interactive map
- **Ideas board** — propose ideas before they're real events; promote them when the group is ready
- **Google Calendar sync** — add finalized events directly to your calendar
- **Circles** — create a group, share an invite code, switch between multiple circles

## Stack

React/TypeScript · Firebase · Claude API · Vercel
